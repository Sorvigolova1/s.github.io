Кинематика
Кинематика-раздел механики, изучающий механическое движение тел.

Механическое движение - перемещение тел в пространстве относительно друг друга с течением времени.

Поступательное - такое движение, при котором любая прямая, проведеннная в теле остается параллельна самой себе. Тело перемещается как единое целое.
Вращательное - такое движение, при котором прямая, проведенная в теле не остается параллельна самой себе. Точки тела движутся по разным траекториям
Движение различают по виду траектории (Прямолинейное, Криволинейное) и по характеру (Равномерное - постоянная скорость, Неравномерное - скорость непостоянная => движение может быть Равноускоренным и Неравноускоренным)
Прямолинейное равномерное движение - такое движение, при котором траектория тела - прямая линия,скорость тела постоянна и за любые равные промежутки времени тело проходит одинаковое расстояние(совершает равные перемещения).

Уравнение движения имеет вид:

x=x0+Vx*t

Формула перемещения:

Sx = Vx*t
  • x-координата тела, м
  • x0-начальная координата тела, м
  • Sx-проекция перемещения,м
  • t-время движения, с
  • Vx-проекция скорости на ось движения,м/с

Прямолинейное равноускоренное движение - движение, при котором скорость тела за любые равные промежутки времени изменяется на одну и ту же величину.

Ускорение-это физическая величена, определяющая быстрому изменения скорости тела.

начало

Формула ускорения:

ax = (Vx-V0x)/t

Формула скорости
при равноускоренном движении:

Vx = V0x + ax*t

Формулы перемещения:

Sx = V0x*t + (axt2)/2

Sx = (Vx2 - V0x2)/2ax

Sx = (V0x + Vx)/2 * t

Уравнение координаты:

x = x0 + V0x*t + (axt2)/2

ax - проекция ускорения, м/c2

V0x - проекция начальной скорости, м/c

Vxпроекция конечной скорости, м/c

t - время движения, с

Sx - проекция перемещения, м

x0 - начальная координата тела, м

x - конечная координата тела, м
